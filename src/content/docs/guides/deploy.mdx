anssl 提供强大的自动部署功能，支持将证书自动部署到各种 Web 服务器和云平台，实现证书管理的完全自动化。

## 配置自动部署

<Steps>

1. **进入部署配置页面**

   访问 [自动部署](https://anssl.cn/console/deploy) 页面

2. **安装自动部署 Cli**

3. **选择证书**

   选择需要自动部署的证书，关联到自动部署。

4. **启用自动部署**

   配置完成后，启用自动部署功能

5. **测试部署**

   建议先执行一次测试部署，确保配置正确

</Steps>
